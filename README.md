### 🚀 Featured Project

**🧠 Mobivia — AI-Based Elderly Mobility Screening**
An award-winning AI desktop app that detects fatigue, disorientation & balance issues in elderly users using **MediaPipe** and **OpenCV**.
- Real-time sit-to-stand detection, tremor tracking & head-stability monitoring
- Responsible-AI design: fully local processing, **no video stored**
- AR-based visual guidance for posture correction
- Built with Tauri + React + TypeScript (team of 5)
- 🏆 **Gold** (INNOVEX 2026) · **1st Runner-Up** (MYVERSE 2026) · **Silver** (i-GEN)
